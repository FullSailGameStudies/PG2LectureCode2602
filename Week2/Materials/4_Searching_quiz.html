C++ Searching Quiz

1.Which of the following is true about linear search?

Correct! Linear search checks each element one by one until it finds the target or reaches the end.
Incorrect.

2. What is the time complexity of binary search in the worst case?

Correct! Binary search has a worst-case time complexity of O(logn) because it halves the search space with each step.
Incorrect.

3. Which condition must be true to use binary search?

Correct! Binary search requires the array to be sorted in order to function correctly.
Incorrect.

4. What is the best-case time complexity of linear search?

Correct! In the best case, the target is the first element, so linear search finds it in constant time.
Incorrect.

5. Why is binary search generally faster than linear search for large datasets?

Correct! Binary search is efficient because it reduces the problem size by half with each comparison.
Incorrect.

6. Which of the following is a disadvantage of binary search?

Correct! Binary search is limited to sorted data, which can be a disadvantage if sorting is expensive or not feasible.
Incorrect.
