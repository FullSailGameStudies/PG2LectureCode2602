C++ Recursion Quiz

1. What is the base case in recursion?

Correct! The base case is the condition that stops the recursive calls and prevents infinite recursion.
Incorrect.

2. What will the following code output?

int factorial(int n) {
    if (n == 0) return 1;
    return n * factorial(n - 1);
}
int main() {
    cout << factorial(3);
}
Correct! factorial(3) = 3 × 2 × 1 = 6
Incorrect.

3. Which of the following problems is best solved using recursion?

Correct! Tree traversal (like in-order, pre-order, post-order) is naturally recursive due to the hierarchical structure of trees.
Incorrect.

4. What is the major drawback of naive recursion in computing Fibonacci numbers?

Correct! Naive recursion recalculates the same Fibonacci values multiple times, leading to exponential time complexity.
Incorrect.

5. What happens if a recursive function lacks a base case?

Correct! Without a base case, the function keeps calling itself until the call stack is exhausted, causing a stack overflow.
Incorrect.
