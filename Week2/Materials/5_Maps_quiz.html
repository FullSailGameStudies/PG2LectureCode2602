C++ std::map Quiz

1. What is the key characteristic of std::map in C++?

Correct! std::map stores elements in sorted order based on the key using a balanced binary search tree.
Incorrect.

2. What is the time complexity of std::map::find() in the worst case?

Correct! std::map is implemented as a Red-Black Tree, so find() operates in logarithmic time.
Incorrect.

3. Which of the following correctly accesses the value associated with a key in a std::map?

Correct! The subscript operator [] is used to access or insert values in a std::map.
Incorrect.

4. What happens if you use map[key] with a key that does not exist?

Correct! Using map[key] with a non-existent key inserts it with a default value (e.g., 0 for int).
Incorrect.

5. Which C++17 feature simplifies iterating over a std::map?

Correct! Structured bindings allow you to unpack key-value pairs directly in the loop header.
Incorrect.

6. What does map.find(key) return if the key is not found?

Correct! If the key is not found, find() returns an iterator equal to map.end().
