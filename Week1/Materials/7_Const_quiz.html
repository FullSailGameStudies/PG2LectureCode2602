C++ Const Quiz

1. What does the 'const' keyword do in C++?

Correct! 'const' declares a variable that cannot be changed.
Incorrect. The correct answer is a) Declares a variable that cannot be changed.

2. How do you declare a constant integer variable in C++?

Correct! Both a) and b) are valid ways to declare a constant integer variable.
Incorrect. The correct answer is c) Both a and b.

3. What is the output of the following code?

const int x = 10;
x = 20;
std::cout << x;
Correct! The code will result in a compilation error because 'x' is a constant and cannot be modified.
Incorrect. The correct answer is c) Compilation error.

4. Can a member function be declared as 'const' in C++?

Correct! A member function can be declared as 'const' to indicate that it does not modify the object.
Incorrect. The correct answer is a) Yes.

5. What is the purpose of 'const' in function parameters?

Correct! 'const' in function parameters prevents the function from modifying the argument.
Incorrect. The correct answer is a) To prevent the function from modifying the argument.

6. What is the output of the following code?

const int x = 5;
const int* p = &x;
*p = 10;
std::cout << *p;
Correct! The code will result in a compilation error because 'p' is a pointer to a constant integer and cannot be used to modify the value.
Incorrect. The correct answer is c) Compilation error.
