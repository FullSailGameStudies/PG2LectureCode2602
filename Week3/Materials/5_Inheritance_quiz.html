C++ Inheritance Quiz

1. What is the main purpose of inheritance in C++?

Correct! Inheritance enables a derived class to reuse and extend the functionality of a base class.
Incorrect.

2. Which keyword is used to define a derived class from a base class?

Correct! The colon : is used in the class declaration to specify inheritance in C++.
Incorrect.

3. What is the order of constructor calls when creating an object of a derived class?

Correct! The base class constructor is always called before the derived class constructor to ensure proper initialization.
Incorrect.

4. What happens if the base class has only a parameterized constructor and no default constructor?

Correct! If no default constructor exists, the derived class must use an initialization list to call the base constructor.
Incorrect.

5. Which access modifier allows public and protected members of the base class to remain accessible in the derived class?

Correct! Public inheritance keeps public and protected members accessible in the derived class with the same access level.
Incorrect.

6. Which of the following is true about constructors in inheritance?

Correct! Constructors are not inherited in C++, but the derived class can call them using an initialization list.
Incorrect.
