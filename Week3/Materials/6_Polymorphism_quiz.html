C++ Polymorphism Quiz

1. Which keyword is used in the base class to enable runtime polymorphism?

Correct! The virtual keyword allows a function to be overridden in derived classes and enables dynamic dispatch.
Incorrect.

2. What does the override keyword do in C++?

Correct! override helps catch errors by ensuring the function matches a virtual function in the base class.
Incorrect.

3. What is the output of the following code?

class Base {
public:
    virtual void show() { cout << "Base"; }
};

class Derived : public Base {
public:
    void show() override { cout << "Derived"; }
};

int main() {
    Base* b = new Derived();
    b->show();
    return 0;
}
Correct! The base class pointer points to a derived object, and show() is virtual, so the derived version is called.
Incorrect.

4. What is the term for the mechanism that determines which function to call at runtime?

Correct! Dynamic dispatch uses the vtable to resolve function calls at runtime based on the actual object type.
Incorrect.

5. Which of the following will cause a compile-time error?

class A {
public:
    virtual void foo() const {}
};

class B : public A {
public:
    void foo() {}  // No const
};
Correct! The function signatures must match exactly, including const. Otherwise, it's not considered an override.
Incorrect.
