C++ Constructors Quiz

1. What is the primary purpose of a constructor in C++?

Correct! Constructors are automatically called when an object is instantiated to initialize its data members.
Incorrect.

2. Which of the following is a valid constructor declaration for a class named Book?

Correct! A constructor has the same name as the class and no return type.
Incorrect.

3. What is a member initialization list used for?

Correct! Member initialization lists are used to directly initialize fields, especially useful for const, reference, and complex types.
Incorrect.

4. Which of the following members must be initialized using a member initialization list?

Correct! const members must be initialized at the time of object creation and cannot be assigned later.
Incorrect.

5. What happens if you do not define any constructor in your class?

Correct! If no constructor is defined, the compiler automatically generates a default constructor.
Incorrect.
