C++ Classes Quiz

1. What is the primary purpose of a class in C++?

Correct! A class encapsulates data and functions, serving as a template for creating objects.
Incorrect.

2. Which access modifier allows class members to be accessed only within the class itself?

Correct! private members are only accessible from within the class, not from outside or derived classes.
Incorrect.

3. What is the default access modifier for class members in C++?

Correct! In C++, class members are private by default unless specified otherwise.
Incorrect.

4. What does the protected access modifier allow?

Correct! protected members are accessible in the class and in any class that inherits from it.
Incorrect.

5. What will happen if you try to access a private member of a class from outside the class?

Correct! Accessing private members from outside the class violates encapsulation and causes a compile-time error.
Incorrect.
