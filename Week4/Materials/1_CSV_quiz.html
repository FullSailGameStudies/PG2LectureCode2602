C++ CSV Quiz

1. What does CSV stand for?

Correct! CSV stands for Comma Separated Values, a format where data fields are separated by commas.
Incorrect.

2. What is the purpose of double quotes in a CSV file?

Correct! Double quotes are used to wrap fields that contain commas, newlines, or quotes.
Incorrect.

3. Which of the following is not a typical limitation of CSV?

Correct! CSV is highly compatible with spreadsheet tools like Excel and Google Sheets.
Incorrect.

4. What happens if a field in a CSV file is left empty?

Correct! Empty fields in CSV are interpreted as missing or null values.
