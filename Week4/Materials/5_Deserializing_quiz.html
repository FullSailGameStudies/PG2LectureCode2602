C++ Deserializing Quiz

1. What is the primary purpose of deserialization in C++?

Correct! Deserialization is the process of converting data from a file (like CSV) into in-memory class instances.
Incorrect.

2. Which C++ standard library class is commonly used to read a CSV file line-by-line?

Correct! ifstream is used to read from files, making it suitable for reading CSV files.
Incorrect.

3. What is the purpose of std::getline(ss, token, ',') in CSV parsing?

Correct! std::getline with a delimiter (,) is used to extract comma-separated values from a line.
Incorrect.

4. Which of the following is the best way to store multiple deserialized objects?

Correct! std::vector is ideal for storing a sequence of objects like deserialized class instances.
Incorrect.

5. What is a potential issue when using std::stoi during deserialization?

Correct! std::stoi throws an exception if the input string cannot be converted to an integer, so error handling is important.
